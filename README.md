# Model Viewer

3D viewer based on [Aka engine](https://github.com/antaalt/Aka).

![Image](asset/screens/sponza.png)

## Features
- Support many model format with Assimp
- Deferred rendering
- Shadow mapping (CSM)
- Directional lights & point lights
- Normal mapping
- PBR rendering
- FXAA
- Skybox
